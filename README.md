# CMS
 angular && webpack

## Getting Started

### Prerequisites
- [Git](https://git-scm.com/)
- [Node.js and npm](nodejs.org) Node ^4.2.3, npm ^2.14.7
- webpack(`npm install -g webpack`)

### Developing
1. Run `npm install` 
2. Run `npm run dev`

## Build 
- develop: `npm run dev`
- product : `npm run build`
